DigiKnit — Voxel Knitter

by Diego Fabri

🖱 Navigation

  • Left Click — place a stitch
  • Shift + Left Click — delete a stitch
  • Click & Drag — paint or erase multiple stitches
  • Right Click + Drag — rotate the camera
  • Mouse Wheel — zoom in / out
  • Ctrl + Z — undo last action
  • + / − grid buttons — resize the voxel grid

🧶 Stitches

  • Knit — standard knit loop, face side
  • Purl — reverse knit loop, purl side
  • Cast On — foundation row stitch

📐 Layers

  • Use ↑ / ↓ Layer buttons to move between rows
  • To build on a second layer, the stitches underneath should be Purls — they provide the loop structure for the next row to anchor into
  • Odd layers automatically flip orientation to match knitting direction

👁 Views

  • Abstract View — switches to knitspeak symbol notation, ideal for knitters who read traditional knitting charts
  • Section View — slice through the model vertically to inspect inner structure

🔁 Circular Knitting

  • Circular knitting mode is included — wrap your rows into a tube or seamless round to simulate knitting in the round

💾 Save, Load & Export

  • Save — stores your model as a .json file
  • Load — reopens a previously saved .json
  • Export — exports the 3D geometry to OBJ and other formats for use in external software

💪 Strength Test

  • Highlights structural weak points based on neighbour connectivity
  • Orange = isolated / pivot risk  ·  Green = well connected
💡 Tip: always start with a Cast On row at layer 0, then switch to Knit for subsequent rows. Place Purls below any stitch that needs to support a layer above.
